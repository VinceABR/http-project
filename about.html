Project Implementation Notes

This website uses one CSS file for all styling. The design includes a consistent menu on every page, Flexbox navigation, CSS Grid content sections, hover effects, borders, padding, margins, colors, and readable font settings.

Milestone #2 adds a JavaScript self-assessment quiz, immediate scoring feedback, and improved visual styling.

This responsive website is a class project for IT 3203: Introduction to Web Development.
